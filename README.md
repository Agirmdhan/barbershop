# Barbershop Management System
# Structure:
# - models/: Model classes (Pegawai, Admin, Barber, Pelanggan, Layanan, Reservasi, Pembayaran, RiwayatLayanan)
# - controllers/: Business logic controllers
# - views/: Streamlit view components
# - utils/: Helper functions and database
# - data/: JSON data storage

## To Run the Application:
# 1. Install dependencies: pip install streamlit pandas
# 2. Run: streamlit run app.py

## Features:
# - Admin Login/Register
# - Barber Management (Create, Read, Update, Delete)
# - Customer Management (Create, Read, Update, Delete)
# - Service Management (Create, Read, Update, Delete)
# - Reservation Management
# - Payment Management
# - Service History Tracking
# - Transaction Reports

## Architecture:
# This application follows MVC (Model-View-Controller) pattern:
# - Models: Define data structures
# - Controllers: Handle business logic
# - Views: Handle UI and user interactions
