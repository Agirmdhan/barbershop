# Assets Folder

## Background Image

Letakkan file background barbershop Anda di folder ini dengan nama `barbershop_bg.jpg` atau `barbershop_bg.png`.

**Rekomendasi:**
- Format: JPG atau PNG
- Ukuran: Minimal 1920x1080px
- File size: < 2MB untuk performa optimal

Jika file tidak ditemukan, sistem akan menggunakan background default (gradient warna).
